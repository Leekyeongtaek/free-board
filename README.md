# 자유 게시판 포트폴리오(작성자 이경택)
## 프로젝트 주제
- 게시글, 댓글, 대댓글 기능을 가진 자유 게시판 프로그램

## 목차
- [프로젝트 상세]
- [요구사항 분석]
- [테이블 ERD]
- [API 기능 소개]

### 프로젝트 상세
- 개발환경 : 인텔리제이
- 자바 : 17
- 스프링부트 : 3.3.1
- MySQL : 8.0.32

### 요구사항 분석
- 회원("/members")
  - 회원가입
- 게시글("/posts")
  - 등록
  - 수정
  - 삭제
  - 상세 조회
  - 목록 조회
  - 좋아요 등록/삭제
- 댓글("/posts/comments")
  - 등록
  - 수정
  - 삭제
  - 대댓글 등록
  - 좋아요 등록/삭제
- 공통
  - 치명적 예외 발생시와 API 응답 시간 1.5초 이상인 경우 관리자에게 알림 메일 전송
